# Config Files

## File Locations
    zshrc         -> $HOME/.zshrc
    starship.toml -> $HOME/.config/starship.toml
    tmux.conf     -> $HOME/.tmux.conf
    vim/          -> $HOME/.vim/
    vim/          -> $HOME/.config/nvim/
    bin/          -> $HOME/.local/bin/

## Installation:

    git clone https://github.com/SolitaryCipher/cfg
    cd cfg
    zsh ./install.sh
